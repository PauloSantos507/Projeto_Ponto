# Ideia de etapas para o projeto:

## Primeira fase:
### 1. Orquestração do Banco de dados:
Utilizei o XAMPP para criar um banco de dados Mysql, chamado sistema_ponto, e nele, criei uma tabela "usuários", que contem 4 colunas, são essas: id, nome, email, senha. Essas colunas receberão os dados de usuário.

### 2. Criação dos arquivos para conexão e configuração:
Feito a criação de arquivos para a configuração com o banco, no momento os arquivos ainda estão sendo configurados.
- Arquivo conexao.php: estrutura a conexao com o banco, realiza uma amostra na tela, evidenciando se a conexão foi ou não bem sucedida.
- login.php: Ponte de conexão entre o front e o banco.

### 3. Criação do Front:
O front utilizará HTML, CSS e JavaScript. Será uma composição básica, mas que para iniciação do projeto será suficiente, e possibilitará upgrades futuros.
